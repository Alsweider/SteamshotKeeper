# SteamshotKeeper
Steam screenshot backup tool. Download your old screenshots.

Download: [Demo version](https://github.com/Alsweider/SteamshotKeeper/releases/latest)

![2025-11-29 14_28_42-SteamshotKeeper 0 0 1 (Demo)](https://github.com/user-attachments/assets/b3b55624-84c2-4a97-9989-0f15bf87699f)

## Usage

1. Enter your Steam name (custom vanity profile name). 
2. Select a destination folder for your screenshots.
3. Click "Download".
4. The programme will download your latest public screenshots on Steam and put them into folders separated by their app-ID.
